# Usage

## From source

First, clone the repo on your machine:

```
$ git clone https://github.com/tryolabs/taggerine.git
$ cd taggerine
```

Finally, use Docker Compose to build and then start the application.

```
$ docker-compose build
$ docker-compose up
```
